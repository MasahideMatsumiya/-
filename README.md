# モリンガミルク エコ割シール

モリンガミルクに貼るエコ割シールのSVGデータセットです。

## シール一覧

| ファイル | デザイン | サイズ |
|---------|---------|--------|
| `sticker_round.svg` | 丸型シール（基本デザイン） | 200×200px |
| `sticker_badge.svg` | バッジ型シール（横長） | 240×120px |
| `sticker_leaf.svg` | 葉っぱ型シール（モリンガ葉モチーフ） | 180×180px |

## デザインコンセプト

- **カラー**: グリーン系（エコ・自然・モリンガを表現）
- **内容**: 「エコ割」「10%OFF」「マイボトル持参割引」
- **モチーフ**: モリンガの葉、リサイクルマーク

## 使い方

SVGファイルをそのまま印刷するか、Inkscape・Illustratorなどで編集してください。

- 割引率を変更したい場合は `10%OFF` のテキスト部分を編集
- 条件テキスト「マイボトル持参割引」は用途に応じて変更可能
- 印刷推奨サイズ: 直径5cm（丸型）/ 6cm×3cm（バッジ型）
